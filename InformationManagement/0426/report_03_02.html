左右マイク間の音声到着のズレ¶

左右のデータの相互相関関数を計算し、その関数が最大値を調べることで、
左右間にある増減関係の時間的なズレを調べる。

相互相関関数を求める関数は次のように定義する。

In [21]:
def calc_cross_corr(x, y):

  num_samples = np.size(y)
  cross_corr = np.zeros(num_samples)

  for m in range(num_samples):
    shifted_y = np.zeros(num_samples)
    shifted_y[:num_samples-m] = y[m:]
    cross_corr[m] = np.dot(x, shifted_y)
    
  return cross_corr

次に、見やすさのため左右のデータを標準化したのち、相互相関関数を実際に計算する。
なお左右間のズレは、
「左のマイクに比べ右のマイクがどれだけ遅れて音を記録するか」
で表現する。

In [22]:
left_norm = (left - np.mean(left)) / np.std(left)
right_norm = (right - np.mean(right)) / np.std(right)


corr = calc_cross_corr(left_norm, right_norm)
corr_norm = corr / np.size(left)

最後に、得られた相互相関関数をプロットするとともに、最大値をとる時間とその値を出力する。

以上の結果から、相互相関係数は 0.000625 seconds の時に最大となることが分かった。
すなわち、左右のマイクの示す音声波形には約 0.000625 秒のずれがあり、同じ時刻に発生した音は左に比べ 0.000625 秒遅れて右に届くということである。

音源方向の推定¶

マイク間の距離を d 、右マイクから左マイクへ向かう方向と音源方向がなす角を θ 、音速を 340 [m/s] としたとき、
同じ音は右マイクには左に比べて τ = d cosθ / 340.0 [sec] 遅れて到着する。
したがって、音源方向 θ は τ を用いて θ = arccos(340.0 τ / d) で求めることができる。
今回用いたマイクの距離は d = 0.3 [m] であり、τ には先ほど得られた時間のズレ τ = 0.000625 [sec] を代入すればよい。

In [24]:
degree = np.rad2deg(np.arccos(340.0 * 0.000625 / 0.3))
print(degree)

44.900527960688976

よって、音源方向は約 44.9 [deg] と推定される。
